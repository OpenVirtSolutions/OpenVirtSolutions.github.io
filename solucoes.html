Nossas Soluções


Saiba mais

O Open Virt Solutions nasceu com o objetivo de oferecer alternativas em software e sistemas open-source, permitindo que nos libertemos da dependência de tecnologias proprietárias. Nossa comunidade é formada por entusiastas apaixonados por soluções livres, sempre dispostos a ajudar e compartilhar conhecimento. No grupo Open Virt Solutions, acreditamos que é possível obter uma solução completa sem custos com licenças ou taxas adicionais para recursos extras, valorizando a liberdade e a colaboração do software aberto.

Proxmox

Proxmox Virtual Environment (PVE) é uma plataforma de virtualização de código aberto baseada em Linux que permite criar e gerenciar máquinas virtuais (VMs) e containers. Ele combina o hipervisor KVM para VMs tradicionais com LXC para containers leves, oferecendo uma interface web intuitiva para administração.

Issabel

Issabel PBX é uma plataforma de comunicação unificada de código aberto baseada no Asterisk, usada para criar centrais telefônicas IP (VoIP). Oferece recursos como ramais, URA, gravação de chamadas, relatórios e integração com sistemas externos, tudo gerenciado por uma interface web. É uma continuação do projeto Elastix, voltada para empresas que buscam uma solução robusta e gratuita para telefonia.

Pfsense

pfSense é um sistema de código aberto baseado em FreeBSD, usado como firewall e roteador. Ele oferece recursos avançados como VPN, balanceamento de carga, controle de banda, NAT, DHCP e monitoramento de rede, tudo gerenciado por uma interface web. É uma solução segura, flexível e gratuita, amplamente usada em redes corporativas e domésticas.

Grafana e Zabbix

Zabbix é uma ferramenta de código aberto usada para monitorar servidores, redes e serviços em tempo real, com alertas automáticos e coleta detalhada de dados. Grafana é uma plataforma de visualização que transforma esses dados em gráficos e dashboards interativos. Juntas, permitem monitoramento completo e visual de ambientes de TI.

OpenMediaVault - OMV

OpenMediaVault é uma solução de armazenamento em rede (NAS) de código aberto que transforma um computador em um servidor para gerenciar e compartilhar arquivos. Com uma interface web fácil de usar, oferece serviços como compartilhamento via SMB, NFS, FTP, suporte a RAID e backups, sendo ideal para uso doméstico e pequenas empresas.
